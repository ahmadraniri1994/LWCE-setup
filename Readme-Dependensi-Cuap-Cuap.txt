LWCE (Lightweight Wayland Compositor Environment).

LWCE terinspirasi dari LXDE dan mencoba mengumpulkan aplikasi atau tool ringan yang bisa digunakan pada lingkungan wayland.

Beberapa tool / aplikasi (sampai saat ini) yang bisa dan mungkin umum digunakan :

- Wayland compositor : Sway (i3 alternatif), Labwc (openbox alternatif), Hikari wm.

- Bar : yambar (Alternatif Lxpanel), yambar lebih ringan daripada waybar.

- Menu : xfce4-appfinder (Alternatif lxpanel menu).

- Terminal : foot (Alternatif xterm, qterminal (alternatif lxterminal).

- Image viewer : lximage (Alternatif gpicview, imv (feh alternatif).

- File manager : pcmanfm-qt (Alternatif pcmanfm).

- Browser : qutebrowser.

- Tools dan aplikasi yang lain bisa ditemukan pada tautan : https://github.com/swaywm/sway/wiki/Useful-add-ons-for-sway
